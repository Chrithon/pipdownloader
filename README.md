# pipinstaller

访问量：![Hits](https://hitcounter.pythonanywhere.com/count/tag.svg?url=https%3A%2F%2Fgithub.com%2FChrithon%2Fpipinstaller)

Pypi包下载器

**介绍：**

三种镜像源可选：豆瓣、清华和USTC。

如果你想用其他的镜像也可以！

**软件需求：**

Python3-pip（Ubuntu环境下）

PyQt5（最好连同把Qt5一起装上）
